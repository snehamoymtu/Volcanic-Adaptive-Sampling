Please download the code and put into the desire working directory. 

Then set the path of the code folder:

Matlab -> Home -> Set Path -> Add with Subfolders -> Save -> Close

This sets the proper paths and saves the path for future Matlab session. 

Then, go to a working directory, and try out the demo :

adaptive_sampling_demo

If you are using this code, please cite our work:

Chatterjee, S., Deering, CD., Waite, GP., Prandi, C., and Lin, P. (2019) An adaptive sampling strategy developed for studies of diffuse volcanic soil gas emissions, Journal of Volcanology and Geothermal Research, https://doi.org/10.1016/j.jvolgeores.2019.06.006
